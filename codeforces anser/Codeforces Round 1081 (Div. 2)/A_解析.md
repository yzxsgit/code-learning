# A. String Rotation Game — 题目解析

---

## 一、题目解析

### 1. 题意理解
给定一个长度为 $n$ 的字符串 $s$，可以对其进行任意次循环旋转（每次将第 $i$ 个字符之前的部分移到末尾）。  
**块（Block）** 定义为：由相同字符组成的、无法向左右延伸的极大连续子串。  
求旋转后字符串的最大块数。

### 2. 核心问题
**关键洞察**：循环旋转相当于选择字符串的任意一个"断点"，将环形字符串从该断点展开成线性字符串。  
块的数量等于相邻字符不同的位置数 + 1（线性情况下）。

旋转只影响**首尾连接处**：若原字符串头尾字符不同，可以通过旋转使这个"额外边界"出现在中间，从而多贡献一个块。

### 3. 解题思路
1. 先统计原字符串中相邻不同的对数，记为 `trans`，初始块数 = `trans + 1`（即 `ans`）。
2. 如果字符串首尾字符不同（`s[0] != s[n-1]`），说明环形上还有一个额外的"变换点"，通过合适的旋转可以把它变成内部边界，块数 +1。
3. 但是如果已经满了（`ans == n`，即所有字符两两不同，已经是最多 $n$ 块），则不能再加。

## 三、关键提示点

### 1. 易错点提醒
- 不需要真正去模拟旋转，只需分析环形上的"相邻不同对"数量。
- 旋转后，环形上的所有相邻对都可能成为内部边界，但线性字符串有 $n-1$ 个内部相邻位置，环形有 $n$ 个。

### 2. 边界条件
- 当 `ans == n` 时（每相邻字符都不同），首尾虽然不同但已经达到最大值，不能再加。
- 字符串长度为 1 时，只有 1 个块，无法旋转出更多。

### 3. 优化思路
- 时间复杂度 $O(n)$，已是最优。

### 4. 常见陷阱
- 误以为需要枚举所有旋转位置（暴力 $O(n^2)$），实际只需 $O(n)$ 分析环形边界。

## 四、代码解析

```cpp
#include <bits/stdc++.h>
using namespace std;

#define ll long long int

void solve(){
    ll n;
    cin >> n;

    string s;
    cin >> s;

    // ans 初始化为 1：任意非空字符串至少有 1 个块
    ll ans = 1;

    // 统计相邻字符不同的位置数（内部边界数）
    // 每个"变换点"额外贡献 1 个块
    for (ll i = 1; i < n; i++)
    {
        if (s[i - 1] != s[i]){
            ans++;   // 发现相邻不同，块数 +1
        }
    }

    // 检查环形上的额外边界：首尾是否不同
    // 条件 1：ans != n 保证还没达到最大块数（避免重复计数）
    // 条件 2：s[0] != s[n-1] 说明旋转后首尾衔接处可以变成内部边界
    if (ans != n && s[0] != s[n - 1]){
        ans++;   // 利用旋转将首尾边界变为内部边界，块数 +1
    }

    cout << ans << "\n";
}

int main() { 
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);
    
    ll T = 1;
    cin >> T;   // 读取测试用例数量
    
    while (T--)
    {
        solve();
    }
    
    return 0;
}
```

### 时间复杂度分析
- **每个测试用例**：$O(n)$，一次线性扫描。
- **总体**：$O(\sum n)$，满足 $n \le 100$，$t \le 500$，远在限制内。

## 五、示例推演

| 输入     | 环形表示          | 环形边界数                  | 答案 |
| -------- | ----------------- | --------------------------- | ---- |
| `abcd`   | a-b-c-d-**a**     | 4处不同                     | 4    |
| `abbc`   | a-b-b-c-**a**     | 3处不同(内)+c≠a(环)         | 4    |
| `abba`   | a-b-b-a-**a**     | 2处不同(内)+a=a(环首尾相同) | 3    |
| `abbccc` | a-b-b-c-c-c-**a** | 3处不同(内)+c≠a(环)         | 4    |
