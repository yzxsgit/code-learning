# B. Flipping Binary String — 题目解析

---

## 一、题目解析

### 1. 题意理解
给定一个长度为 $n$ 的二进制字符串 $s$。每次操作：选择一个下标 $i$，将**除 $i$ 以外**所有位置的字符翻转（0变1，1变0）。每个下标最多被选一次。目标：使所有字符变为 `0`。

### 2. 核心问题
分析操作的本质：  
- **选一个'0'位**做"保护点"：其余所有位翻转 → 等价于：将整个字符串翻转，然后把位置 $i$ 的字符强制复原（它是0，翻转后变1，但你保护了它还是0）。

更简洁的等价理解：**对位置 $i$ 处的字符，选它意味着"此位不翻转，其余都翻转"**。

设操作进行了 $k$ 次，则：
- 每次操作会让大多数位翻转一次。
- **位置 $j$ 处**：如果 $j$ 从未被"选中"，它被翻转了 $k$ 次；如果 $j$ 被"选中"了一次，它被翻转了 $k-1$ 次。

### 3. 解题思路：奇偶分析

设字符串中 `0` 的数量为 $c_0$，`1` 的数量为 $c_1 = n - c_0$。

**策略A**：选所有 `0` 的位置（共 $c_0$ 次操作）  
- 每个 `0` 的位置：被翻转 $c_0 - 1$ 次（其余操作翻转它）。要变为0，需翻转偶数次 → $c_0 - 1$ 为偶数 → $c_0$ 为**奇数**时有效。
- 此时 `1` 的位置：翻转 $c_0$ 次。要变为0，需翻转奇数次 → $c_0$ 为**奇数**时同样有效。✓

**策略B**：选所有 `1` 的位置（共 $c_1$ 次操作）  
- 每个 `1` 的位置：被翻转 $c_1 - 1$ 次。要变为0，需翻转奇数+1=(偶数)次 → $c_1 - 1$ 为奇数 → $c_1$ 为**偶数**时有效。
- 此时 `0` 的位置：翻转 $c_1$ 次。要变为0，需翻转偶数次 → $c_1$ 为**偶数**时有效。✓

**综合**：
- $c_0$ 为奇数 → 选策略A，输出所有 `0` 的下标
- $c_1$ 为偶数（含0）→ 选策略B，输出所有 `1` 的下标
- 否则（$c_0$ 偶，$c_1$ 奇）→ 输出 `-1`

## 三、关键提示点

### 1. 易错点提醒
- 选某位置 $i$ 意味着"其余位翻转"，不是"本位翻转"，容易搞混。
- 当 $c_1 = 0$（全0）时，策略B输出0次操作，是合法的（直接输出0和空行）。

### 2. 边界条件
- 全0字符串：$c_0 = n$，$c_1 = 0$（偶数）→ 策略B，0次操作。
- 全1字符串：$c_0 = 0$（偶数），$c_1 = n$（若 $n$ 为奇数则无解）。

### 3. 常见陷阱
- $c_0$ 为偶且 $c_1$ 为奇 → 无解，两种策略都不成立。
- 两种策略**有可能同时成立**（$c_0$ 奇且 $c_1$ 偶），输出任意一种均可。

## 四、代码解析

```cpp
#include <bits/stdc++.h>
using namespace std;

#define ll long long int

void solve(){
    ll n;
    cin >> n;

    string a;
    cin >> a;

    // 统计字符串中 '0' 的个数
    ll c0 = 0;
    for (ll i = 0; i < n; i++)
    {
        if (a[i] == '0'){
            c0++;
        }
    }

    // '1' 的个数
    ll c1 = n - c0;

    // 策略A：c0 为奇数时，选所有 '0' 的位置
    // 原理：'0'位被翻转 c0-1(偶数)次保持0，'1'位被翻转 c0(奇数)次变为0
    if (c0 % 2) {
        cout << c0 << "\n";           // 操作次数
        for (ll i = 0; i < n; i++)
        {
            if (a[i] == '0'){
                cout << i + 1 << " "; // 输出所有 '0' 的下标（1-indexed）
            }
        }
        cout << "\n";
    }
    // 策略B：c1 为偶数时，选所有 '1' 的位置
    // 原理：'1'位被翻转 c1-1(奇数)次变为0，'0'位被翻转 c1(偶数)次保持0
    else if ((c1 % 2) == 0) {
        cout << c1 << "\n";           // 操作次数
        if (c1 != 0) {               // c1=0 时不需要输出任何下标
            for (ll i = 0; i < n; i++)
            {
                if (a[i] == '1'){
                    cout << i + 1 << " "; // 输出所有 '1' 的下标（1-indexed）
                }
            }
            cout << "\n";
        }
    }
    // 两种策略均不满足（c0偶 且 c1奇）→ 无解
    else{
        cout << -1 << "\n";
    }
}

int main() { 
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);
    
    ll T = 1;
    cin >> T;
    
    while (T--)
    {
        solve();
    }
    
    return 0;
}
```

### 时间复杂度分析
- 每个测试用例：$O(n)$
- 总体：$O(\sum n)$，满足 $\sum n \le 2 \times 10^5$
